# Fonts Directory

Place the `Monogram.ttf` font file in this directory.

The Monogram font is a pixel-style retro font used throughout the Hecate Market application.

## Required File
- `Monogram.ttf` - The main pixel font file

## Usage
The font is loaded via @font-face in the CSS files:
- `public/css/styles.css`
- `public/css/auth.css`
- `public/css/errors.css`
